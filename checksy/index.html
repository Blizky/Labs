How to allow pop-ups and redirects

Checksy opens one tab per enabled site. If your browser blocks these tabs, follow this flow:

  1. Run Check name once.
  2. When a pop-up blocked message/icon appears, click it.
  3. Choose Always allow pop-ups and redirects for this page.
  4. If needed, allow third-party redirects/cross-site access for this page in site settings.
  5. Run Check name again.

Result check: if the page says user not found, page not found, or unavailable, the handle is often available. If a profile page loads, it is usually taken.
